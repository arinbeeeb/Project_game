Название: Игра "Быки и коровы"

Какую проблему решает данный проект: 
Проект демонстрирует ключевые основы алгоритмизации и программирования: генерацию случайных чисел, проверку пользовательского ввода,
использование циклов и условных операторов, работу со строками и списками, а также логику сравнения данных. 
В результате проект помогает закрепить навыки написания структурированного кода, обработки ошибок и создания обратной связи с пользователем.

Целевая аудитория: Студенты, преподователи дисциплин, начинающие разработчики, желающие понять структуру, так же любители 
логических игр.

Цели и задачи моего проекта: 
- Оформить введение проекта
- Сделать Use Case диаграмму и блок схему
- Реализовать код по функционалу игры
- Реализовать код по оформлению игры
- Подготовиться к защите проекта
- Составить и оформить текстовую документацию (ГОСТ)
- Составить и оформить визуальную презентацию (ГОСТ)

Функционал:
- Генерация загаданного числа
- Вывод догадки пользователя
- Подсчет быков и коров
- Вывод результатов хода
- Повторение ходов до победы
